# Les Loups du Nideck — Version Premium

Site Next.js premium (noir & or) pour Natacha Kocher.

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer gratuitement

1. Va sur https://vercel.com
2. Connecte ton GitHub
3. Clique sur **Add Project**
4. Sélectionne ce repo
5. Clique sur Deploy

Ton site sera en ligne en 1 clic.
